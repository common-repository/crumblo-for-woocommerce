Online payments for WooCommerce powered by Crumblo

== Description ==

Crumblo offers integrations with Estonian banks such as Swedbank, SEB, LHV, Luminor, Coop Pank, Citadele and largest global fintechs like Revolut, N26 and Wise. More countries including Latvia, Lithuania and Poland being added in first half of 2022.

Subscriptions start as low as €1,29 /month with no hidden fees.

Total purchase amounts will be transferred directly and, in most cases, instantly from the buyer’s bank account into the seller’s bank account.

Use our [self-service](https://selfservice.forpeeps.eu) to quickly onboard, subscribe, and generate the API key to start accepting payments in less than 10 minutes. Purchase overviews, statistics, additional subscriptions, and refunds will be available from Q3 2022.

Crumblo is a secure payment getaway for e-commerce that utilizes the Open Banking APIs under the EU PSD2 regulation.

More information about our solutions can be found on our [website](https://forpeeps.eu).

== Installation ==

Crumblo for WooCommerce is a free plugin available in wordpress.org plugins directory which you can install directly from your Dashboard.

= Automatic installation =

1. Go to Plugins and select Add new, then go to Search Plugins and type "Crumblo for WooCommerce".
1. Install it by simply clicking "Install Now".
1. When the installation is complete, the "Install Now" button becomes Activate. Click on it to activate the plugin.

= Manual installation =

Another approach is to download a zip file of "Crumblo for WooCommerce" from wordpress.org and upload it to server manually using Dashboard or FTP.

= Configuration =

1. Go to Dashboard, click on WooCommerce in menu, then on Settings.
1. Select tab Payments, look for "Crumblo for WooCommerce" method, click on it (or click on button Manage).
1. Enter all details and click "Save changes".
